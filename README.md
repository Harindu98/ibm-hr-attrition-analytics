## 📌 Project Overview
End-to-end People Analytics project using **Python + XGBoost + Power BI** to predict and prevent employee attrition using the IBM HR Analytics dataset.

## 🎯 Key Results
- ✅ Model Accuracy: **87%+**
- ✅ High Risk Employees Identified: **193**
- ✅ Estimated Replacement Cost: **$7M**
- ✅ Overtime Ban Impact: Risk reduced significantly
